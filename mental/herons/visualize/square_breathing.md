# Square Breathing

```
         name 
     in /    \ pause
      act    quantify
  pause \    / out
        accept 
```

- *re*quantify
  - similar situations comparison
  - time you felt opposite
- acceptance
  - yes i feel **this** emotion
- grounding (examine changes in six senses like umami)
  - see 5
  - feel 4
  - hear 3
  - smell 2
  - taste 1
  - instinct (just go with just do it)
